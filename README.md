# ARM Assembly Mini Toolkit

A small collection of ARM assembly programs for basic embedded-style data processing.

This project demonstrates:
- array traversal
- comparisons
- loops
- arithmetic operations
- memory access
- practical low-level programming patterns

Programs included:
- maximum/minimum search
- average of samples
- threshold counting
- ADC value scaling
- moving average filter

The goal of this project is to build practical ARM assembly fluency for embedded systems and electrical engineering applications.
